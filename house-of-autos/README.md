# House of Autos

Single-page site for **House of Autos** — a private collection of performance cars for self-drive hire in London.

## Structure
```
index.html   the complete site (fonts, images and styles embedded — no external files)
```

`index.html` is fully self-contained: fonts are loaded from Google Fonts and all imagery is embedded, so the folder can be hosted anywhere with no build step.

## Deploy
- **Netlify**: drag this folder onto https://app.netlify.com/drop for an instant URL, or connect the repo.
- **GitHub Pages**: repo Settings → Pages → Source = `main` / root. This site lives at `https://gensynx.github.io/Websitemaker/house-of-autos/`.
- **Vercel**: import the repo, framework preset "Other", no build command.
