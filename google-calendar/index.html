Bookings, straight onto your calendar.

One small script wires any site in this repo to Google Calendar: an appointment-booking embed, add-to-calendar buttons, and live public-calendar views. Copy gcal.js into a site folder, paste a snippet, done — widgets stay invisible until a real link is configured, so nothing breaks on a live site.

In Google Calendar, hit Create → Appointment schedule, set your availability, save. Google handles time zones, clashes and confirmation emails.

04

Attribute reference

Everything is declarative. Add attributes to plain HTML; gcal.js does the rest on load. Site-wide defaults live in window.GCAL_CONFIG (bookingUrl · timezone · calendarId).

Attribute
Meaning
data-gcal-booking-button
Turns the element into a booking button that opens the schedule in an overlay (short links open a new tab). Value = schedule URL, or empty to use the site config. Hidden until a URL exists.
data-gcal-booking data-height
Inline booking embed rendered inside the element. Hidden until configured.
data-gcal-add
Add-to-calendar element. On an <a> the href is set; on anything else a click handler opens the link.
data-title data-start data-end data-duration
Event basics. data-start="2026-08-01T10:00" for timed, "2026-08-01" for all-day (then data-end is the last day, inclusive). No end/duration → 60 minutes.
data-location data-details data-timezone data-recur
Optional extras. data-recur="RRULE:FREQ=WEEKLY;BYDAY=MO" repeats the event.
data-gcal-view data-mode data-weekstart
Live view of public calendar ID(s). data-mode: MONTH · WEEK · AGENDA. data-weekstart="1" starts weeks on Monday.
05

One-time Google setup

A. Booking — appointment schedule

  1. Open calendar.google.com on a computer, signed into the business account.
  2. Press Create → Appointment schedule. Name it, set slot length and weekly availability, save.
  3. Click the schedule on the calendar → Share. Copy the link — or the website-embed code; the wizard reads both.
  4. Paste it into section 01 and ship the generated snippet.
Free Gmail accounts get one appointment schedule; Google Workspace unlocks several plus reminder emails. Bookers never need a Google account.

B. Calendar view — make it public

  1. In Google Calendar, hover your calendar → ⋮ → Settings and sharing.
  2. Under Access permissions for events, tick Make available to public (choose “See only free/busy” to hide event names).
  3. Scroll to Integrate calendar and copy the Calendar ID.
  4. Paste it into section 03.
Add-to-calendar buttons (section 02) need none of this — they're plain links and work for any visitor with a Google account.
