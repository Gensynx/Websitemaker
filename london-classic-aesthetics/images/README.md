# Photo slots

Every image slot on the site currently shows an intentional branded gradient
with a gold line icon. The moment a real photo with the right filename lands
in this folder, the page swaps it in automatically (no code changes needed;
`assets/js/main.js` probes each slot's `data-img` path on load).

Provide JPGs at roughly 2x display size (cards ~1200px wide, editorial
splits ~1400px, page banners ~1900px). Warm, natural light suits the
ivory/gold palette best.
